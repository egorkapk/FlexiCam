FlexiCam is a modular camera system for Unreal Engine that allows you to create camera modes and smoothly transition between them.


- Define camera behavior using independent Camera Modes to achieve dynamic and responsive camera control for  your game.

- Easily blend between different modes within built-in blending functions.

- Integrate with other systems through Gameplay Tags.


Includes ready-to-use templates for Third Person, Top Down, and Fighting-style games.


Documentation link: https://egorkapk.github.io/flexicam-docs/
